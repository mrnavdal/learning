# Mission: Backend Development with Node.js

## Why
Get hired as a fullstack engineer. The user has ~8 years of Flutter (frontend/mobile) experience and strong programming fundamentals — the gap is server-side: building, securing, and deploying backends. Closing that gap makes them hireable as fullstack rather than mobile-only.

## Success looks like
- Can build a REST API in Node.js (HTTP server, routing, middleware) from scratch and explain each layer
- Can design and query a relational database (Postgres/SQL) backing that API
- Can implement authentication (sessions/JWT) and reason about its security tradeoffs
- Can test, containerize, and deploy a backend service
- Can whiteboard/answer backend interview questions confidently (event loop, REST design, DB modeling, auth)

## Constraints
- Strong existing programmer (8y Flutter/Dart): async/await, OOP, HTTP clients, app architecture are known. Don't re-teach programming fundamentals — teach the server-side delta.
- Dislikes learning from videos. Prefers interactive lessons, questions, and quizzes with tight feedback loops.
- Partway through the Zero to Mastery Node.js course (theory portion).

## Out of scope
- Frontend frameworks (React/Vue) for now — the user already has the client-side story via Flutter.
- Non-Node backends (Go, Python, etc.).
