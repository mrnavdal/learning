# learning

Osobní učební repozitář. Jedno téma = jedna složka, každá je samostatný
`/teach` workspace (vlastní `MISSION.md`, lekce, reference).

## Témata

- [`node-backend/`](./node-backend/) — Backend development v Node.js. Cíl: uplatnění jako fullstack engineer.

## Jak to používám

- **Sync počítač ↔ GitHub:** `git pull` na začátku, `git push` na konci práce.
- **Claude Code na telefonu:** otevřu tento repo a pokračuju v učení odkudkoli.
- **Nové téma:** nová složka v rootu + spustit `/teach` v ní.

## Struktura workspace (uvnitř každé složky tématu)

| Soubor / složka | K čemu je |
|---|---|
| `MISSION.md` | Proč se to učím — kotví každou lekci |
| `lessons/*.html` | Interaktivní lekce (otevři v prohlížeči) |
| `reference/*.html` | Cheat sheety k rychlému opakování |
| `assets/` | Sdílené komponenty (styl, kvíz widget) |
| `learning-records/` | Co už umím — řídí, co se učit dál |
| `RESOURCES.md` | Ověřené zdroje (žádná videa) |
| `NOTES.md` | Preference a poznámky lektora |
